## 📖 快速上手

*注: Framix 的使用可能需要一定的命令行知识。如果您不熟悉命令行操作，建议先了解基础的命令行使用方法。*

#### 命令行参数启动
Framix 提供了一系列命令行参数来控制不同的功能和行为。

- **主要命令**
  - `--flick` : 录制并分析视频帧。
  - `--alone` : 仅录制视频。
  - `--paint` : 绘制视频中的分割线条。
  - `--input` : 分析指定的单个视频文件。
  - `--whole` : 分析整个视频文件夹。
  - `--merge` : 聚合多个报告。
  - `--train` : 对视频文件进行归类。
  - `--build` : 构建图像处理模型。

- **附加命令**
  - `--boost` : 快速模式。
  - `--color` : 彩色模式。
  - `--focus` : 视频转换。
  - `--shape` : 图片尺寸。
  - `--scale` : 缩放比例。
  - `--omits` : 忽略区域。

#### 使用示例
- 启动 Framix 并录制分析视频帧：
  ```
  framix --flick --boost --color --focus
  ```

- 仅录制视频，不进行分析：
  ```
  framix --alone
  ```

- 分析整个视频文件夹：
  ```
  framix --whole <视频文件夹路径>
  ```
  
- 分析多个视频文件并应用高级功能：
  ```
  framix --boost --color --focus --omits 0,0,1,0.25 --input <视频文件路径-1> --input <视频文件路径-2>
  ```
  
- 模拟绘制分析区域：
  ```
  framix --paint --shape 350,700 --scale 0.4 --omits 0,0,1,0.2 --omits 0,0.9,1,0.1
  ```

- 为分析完毕的多个视频生成报告：
  ```
  framix --merge <NEXA文件夹路径-1> --merge <NEXA文件夹路径-2> --merge <NEXA文件夹路径-3>
  ```
  
#### 高级功能
- **快速模式 (`--boost`)**: 这个模式加速了视频处理的速度，适合快速预览和测试。
- **彩色模式 (`--color`)**: 用于分析彩色视频，提供更丰富的视觉效果。
- **视频转换 (`--focus`)**: 转换视频格式以适应不同的处理需求。
- **图片尺寸 (`--shape`)**: 自定义视频帧的尺寸和模型的尺寸，以适应不同的分析要求。
- **缩放比例 (`--scale`)**: 自定义图片的缩放比例，以适应不同大小的的屏幕。
- **忽略区域 (`--omits`)**: 指定视频帧中应被忽略的区域，这对于避免分析不必要的部分非常有用。

#### 配置选项
- **新标题 (`header`)**: 为当前视频生成一个新的标题文件夹存放若干视频。
- **新设备 (`serial`)**: 从所有已连接的设备中重新选择一台设备。
- **新参数 (`deploy`)**: 在录制分析模式下修改配置文件以适应不同视频帧的分析策略。
- **读秒数 (`******`)**: 支持最长300秒的录制时长。

#### 多任务处理
Framix 融合了多进程、多线程和协程技术，提供了高效的视频处理能力。这些技术允许 Framix 同时分析多个视频或整个视频文件夹，大幅提升了处理速度。

- **多进程**: 利用 CPU 的多核优势，同时运行多个视频分析任务。
- **多线程**: 在单个进程中同时处理多个任务，优化资源使用和响应时间。
- **协程**: 提供更轻量级的并发处理，尤其适用于 I/O 密集型操作，如文件读写。

#### 交互式前端界面
Framix 提供了一个直观且功能强大的前端界面，帮助用户更容易地管理和分析视频。

- **可视化界面**: 清晰的 UI 设计，使得操作 Framix 更加直观和易懂。
- **手动选择帧**: 用户可以通过前端界面手动选择视频的开始帧和结束帧，以精确控制分析的部分。
- **交互式分析**: 通过前端界面进行，用户可以在界面中选择具体要分析的视频，以及指定开始和结束帧。

#### 多阶段报告展示
Framix 允许用户将视频分析结果划分为多个阶段，便于对特定部分进行详细审查。

- **分阶段展示**: 用户可以将视频分解成多个阶段，例如启动阶段、交互阶段等，每个阶段的分析结果单独展示。
- **自定义阶段**: 根据需要，用户可自定义分阶段的标准和数量。

#### 重点帧展示
Framix 提供了突出显示视频中重要帧的功能。

- **重点帧识别**: 自动识别并突出显示视频中的关键帧，如性能峰值或异常帧。
- **重点帧导航**: 用户可以快速导航至这些重要帧，便于详细分析。

#### 综合报告汇总
Framix 支持将多个视频的分析结果汇总到一个综合报告中。

- **整体视角**: 提供全局视角，将多个视频的分析结果聚合展示。
- **对比分析**: 便于对比不同视频间的性能表现差异。

#### 自动帧差计算
Framix 能自动计算并展示视频帧之间的差异。

- **帧差展示**: 在报告中展示连续帧之间的变化，帮助识别性能波动点。
- **动态调整**: 用户可以根据需要调整帧差计算的精度和范围。

#### 一键复制功能
- **结果复制**: Framix 提供一键复制帧差结果的功能，便于将数据导出或分享。
- **快捷操作**: 提高了操作效率，尤其在需要将结果快速整合到其他文档或报告中时。

#### 性能优化
- Framix 的设计注重性能和速度，即使处理大量视频数据也能保持流畅。
- 前端页面的展示效果优化，确保用户在分析和浏览结果时获得流畅的体验。
